# Changelog

## [0.2.1] - 2026-06-26

### Changed
- 系统托盘图标从单个 CPU 圆环改为 CPU / 内存 / 磁盘 三个同心圆环，与精简模式保持一致。

## [0.2.0] - 2026-06-26

### Added
- 标题栏新增「最小化」按钮。
- 标题栏新增「精简模式」切换按钮，点击后窗口缩小，仅显示 CPU / 内存 / 磁盘 三个同心圆环占用率。
- 新增开机自启管理脚本 `scripts/install_autostart.py` 与 `scripts/remove_autostart.py`。

### Changed
- 托盘图标分辨率从 64×64 提升到 128×128，数字字体更大并增加阴影，任务栏小尺寸下更清晰。
- 精简模式圆环采用与主界面一致的颜色：CPU（蓝/橙）、内存（绿/橙）、磁盘（橙/红）。

### Fixed
- 修复精简模式切换时因缺少 `QPen` 导入导致的异常退出。

## [0.1.0] - 2026-06-26

### Added
- 初始版本：无边框半透明桌面小部件，实时显示 CPU、内存、磁盘、网络状态。
- 系统托盘图标显示 CPU 占用率圆环。
- 支持鼠标拖动、托盘菜单与右键退出。
